<h3>源码简介：</h3>
<p>此套系统互站是1w的，虚拟卡密淘宝天猫代付系统、京东油卡卡密系统、京东中石油、沃尔玛充值、聚合支付系统源码<p>
<p>支持淘宝天猫所有卡密店铺，均可协议回调<p>
<p>系统只有天猫代付和京东中石油模块，截图里京东中石化、比心和快手小店是没有的，只有个名称功能还没有开发<p>
<p>天猫是由获取 ck 软件和使用教程的，但是中石油就没有教程，需要自己研究<p>
